# LoadFileFormat
JPG,PNG等の画像データの細かいファイルフォーマットの中を見てみたかったので、データのバイト列を読み込んでログに出すコードを書きました。
ついでにエディタ拡張にして使えるようにしましたが、多分もっと効率的な方法があると思います。

# Dependencies / ライブラリ依存
DotNetZip(Ionic Zip Library)
